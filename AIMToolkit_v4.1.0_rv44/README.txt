This is an official release of the AIM Toolkit version 4.1.0.
The code is approved for release on November 22, 2013. 

The toolkit consists of a model, documentation and source code 
of the AIM library. This release supports AIM 4.0 model. The 
release contains AIM 4.0 schemas, documentation and CMake-enable 
code for the library.

There are two folders, which are doc and source.

The doc folder consists of AIM-related documents and XML schemas.
*Note: user should use AIM_v4_rv44_XML.xsd for creating and
validating AIM XML documents.

The source folder contains C++ source code for building AIM 
library and ANNIVATR conversion tool.
